<!-- Plugin description -->
This plugin provides MyBatis supports for IntelliJ IDEA.

## Features
- Java Code and XML Mapper Navigation
- XML Mapper `sql`, `include`, `parameterMap` and `resultMap` Navigation
- SQL Log Catching
- XML Mapper and Configuration File Templates
- Kotlin Support
